Assignment: Intro to Verilog with CircuitVerse – 3-Input AND Gate

Goal: In this assignment, you will use the online digital logic simulator CircuitVerse to create a 3-input AND gate using only the standard 2-input AND gates available in the editor. You will then export the generated Verilog file and upload it to Canvas, where an automated testbench will verify correctness.

Learning Objectives

  • Become familiar with CircuitVerse’s logic design environment.
  • Construct a 3-input AND gate using multiple 2-input AND gates.
  • Properly label all inputs and outputs for auto-grading.
  • Export and submit the Verilog representation of your circuit.

Part 1 – Create a CircuitVerse Account and New Project

  1. Go to https://circuitverse.org.
  2. Create a free account or log in.
  3. Click New Circuit to start a fresh design.
  4. Name the project three_input_and or something descriptive.

Part 2 – Build a 3-Input AND Gate Using 2-Input AND Gates

Because CircuitVerse only provides 2-input AND gates, you will combine them to create a 3-input AND gate.

  1. In the CircuitVerse toolbox, add:
    • Three (3) input pins – these will be your inputs.
    • Two (2) 2-input AND gates.
    • One (1) output pin.
  2. Construct the 3-input AND function:
    • Connect input a and b to the first AND gate.
    • Connect the output of that AND gate into one input of the second AND gate.
    • Connect input c to the second input of the second AND gate.
    • Connect the output of the final AND gate to your output pin.
  3. Test your design by toggling inputs:
    • Only when a = 1, b = 1, and c = 1 should the output be 1.
    • All other combinations should yield output 0.

Part 3 – Label Inputs and Outputs

You must use the following exact signal names for the auto-grader:

  • Input 1: a
  • Input 2: b
  • Input 3: c
  • Output: y

To label pins in CircuitVerse:

  1. Select each input pin and rename them to a, b, and c.
  2. Select the output pin and rename it to y.

Your exported Verilog should follow the same interface:

module and_gate(a, b, c, y);
    input  a;
    input  b;
    input  c;
    output y;

    assign y = a & b & c;
endmodule

Important: Your module name must be and_gate and the ports must be named a, b, c, and y. The auto-grader depends on these exact names.

Part 4 – Export Verilog from CircuitVerse

  1. Open the project menu inside CircuitVerse.
  2. Select Export → Verilog.
  3. Download the generated Verilog file.
  4. Rename the file to and_gate.v before uploading.
  5. Open the file and verify:
    • Top-level module is named and_gate.
    • Ports are a, b, c, and y.
    • The functionality correctly implements a 3-input AND.

Part 5 – Automated Testbench & Grading

An automated testbench will evaluate your design by applying all eight possible input combinations:

  • 000 → 0
  • 001 → 0
  • 010 → 0
  • 011 → 0
  • 100 → 0
  • 101 → 0
  • 110 → 0
  • 111 → 1

Your submission must compile cleanly and produce the correct results during simulation.

What to Submit on Canvas

  1. Your Verilog file named and_gate.v.
  2. The module inside must be named and_gate with ports a, b, c, and y.

Tips & Common Issues

  • Spelling matters! Wrong port names will cause auto-grading to fail.
  • Use exactly two AND gates. Do not use gates other than 2-input AND gates.
  • Confirm labels. If your input pins are still named “In1” or similar, fix them before exporting.
  • Only one module. Ensure your Verilog file contains a single top-level module.

Once finished, upload and_gate.v to Canvas to be graded automatically.
